Welcome!

This is a simple FastAPI application with user authentication using OAuth2 and JWT.

You can use this as a starter template to build secure APIs.

How it Works

At first, the user must authorize their account using their username (first name) and password. Only users who have a valid Google Client ID and Client Secret will be authorized. If you don't have these credentials, you can create them from your Google Cloud Console.

All user data is securely stored in a MySQL database and fetched dynamically during authentication.
